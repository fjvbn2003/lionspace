1주차

두근두근 설레는 첫만남

ICE BREAKING
  • 다음주차부터 6주차의 스터디 방향 제시 - 하나의 주제를 정하여 공부, 각자공부, 한 주차마다 팀원1명이 발표자료 준비 후 발표 등...
  • 스터디 기간 중 참여가 힘든 해외여행 및 특별한 일정 공유
  • 학생을 뽑는 방식 및 활동 했던 기수의 활동방식 공유
  • 팀원 박예은 -> 프론트 위주의 스터디 자료준비 - 그 외 팀원 -> 백 or 학생들에게 알려줄 수 있는 꿀팁 자료준비

2주차

박상범, 박예은 발표

새로운 환경을 무서워하지 마라(박상범 발표)
  • 이제 시작하는 컴퓨터 언어도 새로운 언어이다
  • 많은 언어 중, 파이썬을 할 것이며, 프레임워크로는 장고를 사용할 것이다.
  • 프로그래밍은 컴퓨터 언어로 생각을 구체화 해서 나타내는 것

HTML 및 CSS 의 간단한 소개(박예은 발표)
  • HTML과 CSS + a 로 앞으로 프론트엔드를 할 것이다.
  • HTML은 사람의 얼굴뼈같은 틀, CSS는 피부와 외모 생김새 등의 꾸밈

3주차

김영주 발표

P5로 시작하는 Javascript
  • Document Tree
  • 이벤트 핸들링(버튼, 마우스, 키보드 이벤트 및 콜벡함수)
  • 네트워크 (Ajax, Socket)
  • 다른 JavaScript 라이브러리(React, Chart js) 소개
  • Node.js (서버 사이드 자바스크립트)
  • P5.js로 자바스크립트 공부하기
  • 실습(원 그리기, 공 튀기기)
